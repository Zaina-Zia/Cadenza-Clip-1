---------------------------
PROJECT OVERVIEW
---------------------------
This project focuses on analyzing lyric intelligibility using the dataset provided in the Cadenza challenge.  
The notebook performs exploratory data analysis (EDA) on audio and metadata files to understand data 
distributions, extract key features (e.g., MFCCs, spectral measures), and visualize trends related to intelligibility.

HOW TO USE
---------------------------

1. Install dependencies:
   pip install -r requirements.txt

2. Open the notebook:
   jupyter notebook Assignment_1.ipynb

3. Run cells sequentially to reproduce results.

---------------------------
NOTES
---------------------------
- Place all required dataset files inside the `data/` directory before running the notebook.
- Generated plots will automatically be saved in the `plots/` folder.
- The `requirements.txt` lists all Python libraries used for audio analysis and visualization.
